# Experimental Physics Exam

Welcome to the **Experimental Physics Exam Preparation** repository. This project contains notes, concepts, and summarized chapters used for studying for the Semester 4 Experimental Physics exam.

## Repository Structure

The knowledge base is structured as an Obsidian vault to allow for interconnected notes and seamless navigation. 

### Key Directories
- `concepts/`: Contains detailed, granular notes on specific physics topics, organized for easy review.
- `skills/`: Agent and system skills for processing and managing the knowledge base.
- Root Markdown files: Include processed and raw chapter summaries (`Chapter6-Processed.md`, etc.).

## Main Topics Covered

This repository covers a wide range of experimental physics topics, including but not limited to:
- **Vacuum Technology:** Vacuum generation, pumps, materials, and pressure measurement.
- **Cryogenics:** Liquefaction of gases, cryostats, low-temperature cooling, and superfluidity.
- **Thermometry:** Radiation thermometry, resistance thermometers, and precision temperature measurements.
- **Optics & Spectroscopy:** Lasers, LEDs, and atomic/molecular spectra.
- **Particle & Photon Detection:** Principles and applications of various detectors and sources.

## How to use

This repository functions best when opened with [Obsidian](https://obsidian.md/), allowing you to utilize wikilinks, back-references, and graph views.
For an overview of the concepts and how they interconnect, start by viewing the `Mind-Map.md` file in Obsidian.
